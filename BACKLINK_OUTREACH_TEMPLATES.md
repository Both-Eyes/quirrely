## Best Practices

1. **Personalize every email.** Generic pitches get deleted.
2. **Reference their specific work.** Show you've read them.
3. **Keep it short.** Busy people skim.
4. **Follow up once.** After 5-7 days if no response.
5. **Offer value first.** The link should be secondary.
6. **Track everything.** Use the tracking table above.
