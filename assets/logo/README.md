# Sentense Logo Assets

**Design:** Version B — Cradling at Chest  
**Mascot:** Leucistic Squirrel holding a coral treasure

## ⚠️ IMPORTANT: Accurate Leucistic Features

This logo is based on **leucistic** squirrel characteristics (NOT albino):

| Feature | Leucistic (✅ Our Design) | Albino (❌ NOT Used) |
|---------|---------------------------|----------------------|
| **Eyes** | DARK (brown/black) | Pink/red |
| **Nose** | DARK (normal pigmentation) | Pink |
| **Skin/Ears** | Normal pigmentation | Pink |
| **Fur** | White/cream | Pure white |

The key identifier is **dark eyes** — leucistic animals retain normal eye pigmentation.

## Key Design Features

1. **DARK eyes** — The key leucistic trait (NOT pink/red like albino)
2. **DARK nose** — Normal pigmentation (NOT pink like albino)
3. **Cream white fur** — Leucistic coloring
4. **Clear arms** — Visibly cradling the treasure
5. **Coral treasure** — Represents collected sentences/words
6. **Fluffy tail** — Signature squirrel feature
