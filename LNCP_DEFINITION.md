# LNCP — Definition (LOCKED)

## Official Name

**LNCP = Lexical-Numerical Cognitive Protocol**

## Definition

A lightweight framework for analyzing and categorizing communication patterns in written text using rule-based linguistic analysis.

## Components

| Term | Meaning |
|------|---------|
| **Lexical** | Word-level patterns, vocabulary, phrasing |
| **Numerical** | Quantitative metrics (sentence length, ratios, counts) |
| **Cognitive** | Mental patterns expressed through writing style |
| **Protocol** | Standardized specification for analysis |
