# Quirrely LNCP Profile Manifest v3.1.3

## The 10 Core Writing Profiles

| Profile | Key | Description |
|---------|-----|-------------|
| ASSERTIVE | assertive | Direct, confident voice with strong declarative statements |
| HEDGED | hedged | Measured, qualified voice acknowledging uncertainty |
| CONVERSATIONAL | conversational | Informal, accessible voice like speaking aloud |
| FORMAL | formal | Structured, authoritative voice following conventions |
| DENSE | dense | Information-rich, compressed voice with high concept load |
| MINIMAL | minimal | Sparse, economical voice — every word earns its place |
| POETIC | poetic | Lyrical, image-driven voice with heightened rhythm |
| ANALYTICAL | analytical | Systematic, evidence-grounded voice (formerly PARALLEL in early specs) |
| INTERROGATIVE | interrogative | Question-driven voice that opens rather than closes |
| LONGFORM | longform | Expansive, elaborative voice suited to extended argument |

## Note on Naming
Early architecture docs referenced BALANCED or PARALLEL as the 10th profile.
The canonical name is **ANALYTICAL** as defined in lncp/engine/profiles.py StyleAxis enum.
